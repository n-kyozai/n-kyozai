005 日英関係史（年表）

日英関係史（江戸〜幕末期）
1600年慶長5年
オランダ船リーフデ号が漂着 → 英人ウィリアム・アダムズ (三浦按針)が徳川家康の外交顧問に。
1613年慶長18年
イギリスが平戸に商館を開設 (1623年閉鎖)
1808年文化5年
英軍艦が長崎湾内に侵入、薪水・食料を強奪 (フェートン号事件)
1824年文政7年
イギリス捕鯨船が常陸大津浜や薩摩宝島で暴行事件
1825年文政8年
幕府、異国船打払令 (無二念打払令) を出す
1854年安政元年
長崎で日英和親条約に調印
1858年安政5年
日英修好通商条約調印。
初代駐日総領事 (のち公使) オールコック来日。
※貿易開始後、英は国別貿易取扱高の8割以上を占めるようになる(1865年)。
1861年文久元年
攘夷派が英仮公使館の東禅寺を襲撃 (東禅寺事件)
1862年文久2年
生麦事件 (薩摩藩士がイギリス人を殺傷) → 翌年の薩英戦争へ
1863年文久3年
英公使館焼打ち事件 (長州の高杉晋作ら)。
長州藩外国船砲撃事件。
1864年元治元年
四国艦隊下関砲撃事件 (英・仏・米・蘭が長州を攻撃)
1865年慶応元年
駐日公使パークス来日、薩長などの討幕勢力を支援
日英関係史（明治〜大正期）
1886年伊藤博文①
ノルマントン号事件 (英船長が日本人乗客を見殺しにした事件)
→ 領事裁判による軽い判決で、国民は法権回復(条約改正)の必要性を痛感した。
1894年伊藤博文②
ロンドンで駐英公使青木周蔵が日英通商航海条約に調印。
→ 領事裁判権の廃止、関税自主権の一部回復など。（当時の外相は陸奥宗光）
1902年桂太郎①
ロシアの南下を警戒し、日英同盟協約締結。
1905年
第二次日英同盟締結 → 日本の韓国保護国化を承認し、適用範囲をインドにまで拡大。
1911年桂太郎②
第三次日英同盟締結 → ドイツの進出に対応し、アメリカを適用外とした。
1914年大隈重信②
第一次世界大戦勃発 → 日本、日英同盟を理由に参戦（対独宣戦布告）。
1921年高橋是清
ワシントン会議にて四カ国条約調印 → これに伴い日英同盟は廃棄された(1923年)。
日英関係史（昭和：戦前〜戦中）
1925年加藤高明①
上海の反日・反英デモに英警察が発砲 (五・三〇事件)
→ 中国で反帝国主義運動が活発化。
1930年浜口雄幸
ロンドン海軍軍縮条約締結（補助艦の制限）。
野党などが海軍軍令部の反対を押し切った調印を非難（統帥権干犯問題）。
1932年犬養毅 / 斎藤実
国際連盟が満州事変調査のためイギリスのリットンを団長とする調査団を派遣。
世界恐慌を受け、イギリスがブロック経済圏（スターリング・ブロック）を構築。
1941年東条英機
日本包囲網（ABCD包囲陣）の形成。12月、英米に宣戦布告し太平洋戦争開戦。
重要事項と補足解説

  • 江戸幕府の異国船対策の変遷


    対外的な危機感の高まりや国際情勢の変化により、幕府の対応は二転三転しました。
    年・法令名 内容と背景
    1806年
    文化の撫恤令
    （薪水給与令）    		 ロシア船の来航などを受け、穏便に薪水（食料・水など）を与えて帰らせる方針をとった。
    1825年
    異国船打払令
    （無二念打払令）    		 イギリス捕鯨船の暴行事件などが頻発したため、問答無用で外国船を打ち払う強硬な方針に変更。
    1842年
    天保の薪水給与令    		 清がアヘン戦争でイギリスに敗れたことを知り、外国との紛争を避けるため再び柔軟路線（漂着船への給与）へ転換。

  • 不平等条約改正への歩み


    幕末に結んだ不平等条約（領事裁判権を認め、関税自主権がない）の改正は、明治政府の悲願でした。
    外相名 主な出来事と結果
    井上馨 鹿鳴館での欧化政策。外国人判事の任用案が反対に遭い、ノルマントン号事件の国民的怒りもあって辞任。
    大隈重信 外国人判事を大審院のみに限定する案で交渉。ロンドン・タイムズ紙の報道で批判を浴び、爆弾テロで負傷し辞任。
    陸奥宗光 日清戦争直前の1894年、日英通商航海条約に調印。領事裁判権の完全撤廃に成功。
    小村寿太郎 1911年、日米通商航海条約などで関税自主権の完全回復に成功し、条約改正が完了。

  • 幕末の密航と留学生


    【長州五傑（長州ファイブ）】
    1863(文久3)年、長州藩の伊藤博文や井上馨らがイギリスへ密航留学しました。攘夷を掲げる藩にあって命がけの渡航でしたが、彼らは西洋の技術や国力に圧倒され、開国・近代化へと思想を転換させました。
    【薩摩藩の留学生】
    1865(慶応元)年、薩摩藩の五代友厚らもイギリスへ密航しました。幕府が海外への渡航を正式に認めたのは、その翌年の1866年のことです。
