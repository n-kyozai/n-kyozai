数学 基礎公式集

数と式・数列・図形・データの重要事項

1. 数と式

Point Check: 基本公式

指数法則・展開
• $(a + b)c = ac + bc$
• $a^m \times a^n = a^{m+n}$
• $(a^m)^n = a^{mn}$
• $(a + b)^2 = a^2 + 2ab + b^2$
• $(a + b)(a - b) = a^2 - b^2$
重要: 展開公式は因数分解の逆。どちら方向も即座に変換できるように！
因数分解・有理化
• 共通因数でくくる: $ax + ay = a(x + y)$
• たすき掛け: $acx^2+(ad+bc)x+bd = (ax+b)(cx+d)$
• 分母の有理化：
$\displaystyle \frac{c}{\sqrt{a}+\sqrt{b}} = \frac{c(\sqrt{a}-\sqrt{b})}{a-b}$
約数の個数
素因数分解が $N = p^a \times q^b \times \dots$ のとき
• 約数の個数 $= (a+1)(b+1)\dots$
例: $12 = 2^2 \times 3^1$ なので $(2+1)(1+1)=6$個。

2. 数列

Point Check: 数列の公式と規則性

等差数列
初項 $a$, 公差 $d$, 末項 $l$ のとき
• 一般項 ($n$項)： $a_n = a + (n - 1)d$
• 和 ($S_n$)：
$\displaystyle S_n = \frac{1}{2}n (a + l)$
$\displaystyle S_n = \frac{1}{2}n \{ 2a + (n - 1)d \}$
自然数の和 (Σ公式)
• $1$から$n$までの和：
  $\displaystyle \sum_{k=1}^n k = \frac{1}{2}n(n + 1)$
• $1$から$2n-1$までの奇数の和：
  $\displaystyle 1+3+5+\dots+(2n-1) = n^2$
裏技: 奇数の和は「項数の2乗」になる！（例: 1+3+5 = 3項なので $3^2=9$）
規則性の問題 (タイル等)
図形が増えていく問題は、「前の項との差（階差）」や「n番目の式」を見つけることが鍵。
例: $1, 3, 6, 10\dots$ と増えるなら、
増え方は $+2, +3, +4\dots$ となる。

3. 図形と計量

Point Check: 面積・体積・三角比

平面図形（扇形）
半径 $r$, 中心角 $a^\circ$ の扇形
• 弧の長さ $l = 2\pi r \times \frac{a}{360}$
• 面積 $S = \pi r^2 \times \frac{a}{360} = \frac{1}{2}lr$
空間図形（球・錐）
半径 $r$ の球
• 表面積 $S = 4\pi r^2$ (心配ある事情)
• 体積 $V = \frac{4}{3}\pi r^3$ (身の上に心配ある)

底面積 $S$, 高さ $h$ の錐体 (円錐・角錐)
• 体積 $V = \frac{1}{3}Sh$
錐体の体積は「柱の3分の1」と覚える！
三角比の相互関係
• $\tan\theta = \frac{\sin\theta}{\cos\theta}$
• $\sin^2\theta + \cos^2\theta = 1$

4. データの分析

Point Check: 用語まとめ

代表値と散らばり
平均値 ($\bar{x}$)データの総和 $\div$ データの個数
中央値 (メジアン)データを大きさ順に並べたときの中央の値。
偶数個の場合は中央2つの平均。
最頻値 (モード)データの中で最も頻繁に現れる値。
範囲 (レンジ)最大値 - 最小値
四分位数データを4等分する位置の値。
第1($Q_1$), 第2(中央値), 第3($Q_3$)。
四分位範囲$Q_3 - Q_1$ (データの真ん中50%の範囲)
分散と標準偏差
• 偏差： (データ - 平均値)
• 分散 ($s^2$)： 偏差の2乗の平均値。
$\displaystyle s^2 = \frac{1}{n} \sum (x_i - \bar{x})^2$
• 標準偏差 ($s$)： 分散の正の平方根 ($\sqrt{\text{分散}}$)。
単位が元のデータと同じになるため扱いやすい。
