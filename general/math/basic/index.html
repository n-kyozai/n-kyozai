基礎問題

数学 基礎公式集

数と式・図形・データ・数列の重要事項まとめ

数と式

指数法則・展開
• $(a + b)c = ac + bc$
• $a^m \times a^n = a^{m+n}$
• $(a^m)^n = a^{mn}$
• $(a + b)^2 = a^2 + 2ab + b^2$
• $(a + b)(a - b) = a^2 - b^2$
因数分解・有理化
• 共通因数でくくる: $ax + ay = a(x + y)$
• たすき掛け: $acx^2+(ad+bc)x+bd = (ax+b)(cx+d)$
• 分母の有理化：
$\displaystyle \frac{c}{\sqrt{a}+\sqrt{b}} = \frac{c(\sqrt{a}-\sqrt{b})}{a-b}$
約数
素因数分解が $p^a \times q^b \times \dots$ のとき
• 約数の個数 $= (a+1)(b+1)\dots$
時短テクニック:
約数の個数を聞かれたら、素因数分解して「指数+1」を掛け算！
例: $12=2^2 \times 3^1 \to (2+1)(1+1)=6$個

図形と計量

平面図形（扇形）
半径 $r$, 中心角 $a^\circ$ の扇形
• 弧の長さ $l = 2\pi r \times \frac{a}{360}$
• 面積 $S = \pi r^2 \times \frac{a}{360} = \frac{1}{2}lr$
空間図形（球・錐）
半径 $r$ の球
• 表面積 $S = 4\pi r^2$ (心配ある事情)
• 体積 $V = \frac{4}{3}\pi r^3$ (身の上に心配ある)

底面積 $S$, 高さ $h$ の錐体
• 体積 $V = \frac{1}{3}Sh$
Point:
円錐や角錐の体積は、必ず柱の体積の「3分の1」になる！
三角比の相互関係
• $\tan\theta = \frac{\sin\theta}{\cos\theta}$
• $\sin^2\theta + \cos^2\theta = 1$

データの分析

代表値と散らばり
平均値 ($\bar{x}$)データの総和 $\div$ データの個数
中央値 (メジアン)データを大きさ順に並べたときの中央の値。
偶数個の場合は中央2つの平均。
最頻値 (モード)データの中で最も頻繁に現れる値。
範囲 (レンジ)最大値 - 最小値
四分位数データを4等分する位置の値。
第1($Q_1$), 第2(中央値), 第3($Q_3$)。
四分位範囲$Q_3 - Q_1$ (データの真ん中50%の範囲)
分散と標準偏差
• 分散 ($s^2$)：
$\displaystyle s^2 = \frac{1}{n} \sum (x_i - \bar{x})^2$
(偏差の2乗の平均)

• 標準偏差 ($s$)：
$\displaystyle s = \sqrt{s^2} = \sqrt{\text{分散}}$
意味:
標準偏差が大きいほど、データの散らばりが大きい（平均から離れたデータが多い）。

数列

等差数列
初項 $a$, 公差 $d$, 末項 $l$
• 第$n$項： $a_n = a + (n - 1)d$
• 和： $\displaystyle S_n = \frac{1}{2}n \{ 2a + (n - 1)d \}$
  または $\displaystyle S_n = \frac{1}{2}n (a + l)$
イメージ:
等差数列の和は「(最初 + 最後) × 個数 ÷ 2」と覚える！
自然数の和 (Σ公式)
• $1$から$n$までの和：
  $\displaystyle \sum_{k=1}^n k = \frac{1}{2}n(n + 1)$
• 奇数の和 ($1, 3, 5\dots$)： $n^2$ (項数$n$の場合)
