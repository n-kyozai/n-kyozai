003 日米関係史（年表）

日米関係史（幕末〜明治期）
1837年
モリソン号事件（異国船打払令(1825)に基づく）
1846年
ビッドル、浦賀に来航して通商要求 → 幕府は拒否
1853年
ペリー、浦賀に来航
1854年
ペリー再来航 → 日米和親条約締結（片務的最恵国待遇）
1856年
ハリス、初代駐日総領事として下田に着任
1858年
日米修好通商条約締結（米に領事裁判権・日の関税自主権欠如）
1860年
通商条約批准書交換の遣米使節派遣（咸臨丸が随伴）
駐日米公使館の通訳ヒュースケン、攘夷派に暗殺される
1864年
四国艦隊下関砲撃事件（英・仏・米・蘭）
1866年
英・仏・米・蘭との間で改税約書調印 → 日本の輸入関税引下げ
1871年
岩倉遣外使節団が出発、米欧諸国を訪問 (~1873)
1876年
開拓使の招きでクラークが札幌農学校に着任
1878年
外務卿寺島宗則の関税自主権回復交渉に米が同意 → 英・独の反対で不成立。
フェノロサ、東大に招かれて来日 → 日本美術の復興に尽力。
1894年伊藤博文②
英に続き日米通商航海条約に調印 → 領事裁判権撤廃
1898年大隈重信①
米、ハワイとグアム・フィリピンを領有。太平洋進出本格化。
1899年山県有朋②
国務長官ジョン=ヘイが中国に関して「門戸開放宣言」を発表（門戸開放・機会均等・領土保全の3原則）
1905年桂太郎①
米大統領T=ローズヴェルト、ポーツマス講和会議を仲介。
米鉄道王ハリマンの満鉄共同経営案を日本が拒否。
桂・タフト協定（日: 韓国指導権、米: フィリピン統治）
1906年西園寺公望①
カリフォルニアなどで日本人移民排斥問題起こる
1911年桂太郎②
改正日米通商航海条約に調印 → 関税自主権を回復
日米関係史（大正〜戦中）
1914年大隈重信②
第一次世界大戦勃発 (~1918、米は1917年に参戦)
1917年寺内正毅
中国に関する石井・ランシング協定締結
1918年
ウィルソン大統領が「14ヵ条の平和原則」発表
1920年原敬
国際連盟設立。米は上院の反対で加盟せず。
1921年・1922年原敬 / 高橋是清
ワシントン会議開催。四カ国条約(1921)・九カ国条約・ワシントン海軍軍縮条約(1922) 締結。
1923年加藤友三郎
九カ国条約により石井・ランシング協定廃棄
1924年清浦奎吾
米、排日移民法を制定
1927年田中義一
米大統領クーリッジの提案でジュネーヴ軍縮会議 → 不成功
1928年
米仏の提案で、パリで15ヵ国が不戦条約締結
1929年浜口雄幸
世界恐慌: 米から全資本主義世界に波及
1930年
補助艦制限に関するロンドン海軍軍縮条約に調印 (英・米・日)
1931年若槻礼次郎②
満州事変 (~1933) → 米は日本の一連の行動を不承認
1936年岡田 / 広田
ロンドン・ワシントン両海軍軍縮条約失効
1937年近衛文麿①
日中戦争勃発
1939年平沼騏一郎
第二次世界大戦始まる。米、日米通商航海条約廃棄を通告 (翌年失効)。
1940年近衛文麿②
日独伊三国軍事同盟締結 → 米、屑鉄・鉄鋼の対日禁輸
1941年近衛③ / 東条
日米交渉 (野村吉三郎駐米大使とハル国務長官)。
日本、南部仏印進駐 → 米、在米日本人資産凍結、石油の対日禁輸。
米、ハルノートで最終提案。
日本、ハワイの真珠湾を攻撃 (12.8) → 太平洋戦争
1942年
ミッドウェー海戦で日本敗北 → 米に戦局の主導権
1943年
カイロ宣言発表（米: F=ローズヴェルト、英: チャーチル、中: 蒋介石）
内容: 日本の植民地・占領地の剥奪、対日戦争遂行の確認。
1945年小磯 / 鈴木貫
ヤルタ会談（米・英・ソ）→ 対独戦後処理。秘密協定でソ連の対日参戦。
ポツダム会談（米: トルーマン、英: アトリー、ソ: スターリン）
→ 日本の無条件降伏要求、占領、武装解除、領土限定などを宣言。日本は受諾(8.14)。
日米関係史（戦後〜現代）
1945年東久邇 / 幣原
GHQ（最高司令官マッカーサー）による占領政策始まる。
五大改革指令・神道指令・農地改革・財閥解体・公職追放等の指令。
1946年
極東国際軍事裁判開廷 (~1948)
1948年吉田茂②
GHQ、経済安定九原則を指示
1949年吉田茂③
ドッジラインの実施、シャウプ税制改革勧告。
マッカーサー「日本は反共の防壁」と言明。
1950年
GHQ、レッド・パージを指令。
朝鮮戦争勃発。マッカーサー、警察予備隊創設を指示。
1951年
サンフランシスコ平和条約調印（日本と連合国48ヵ国）。
日米安全保障条約調印（翌年発効）。
1952年
日米行政協定調印・発効
1953年吉田茂⑤
奄美諸島返還
1954年
米のビキニ環礁水爆実験で第五福竜丸が被爆。
MSA協定（日米相互防衛援助協定） → 防衛庁・自衛隊発足。
1957年岸信介①
岸信介首相・アイゼンハワー米大統領「日米新時代」を強調
1960年岸信介②
日米相互協力及び安全保障条約（新安保条約）・日米地位協定調印 ← 安保闘争
1968年佐藤栄作②
小笠原諸島返還
1969年・1970年佐藤栄作③
佐藤・ニクソン会談で沖縄返還に合意(1969)。
日米安保条約自動延長入り（70年安保闘争）。
1971年
沖縄返還協定調印(翌年返還)。
ニクソン大統領、訪中計画と金・ドル交換停止発表（ニクソン・ショック）。
1975年・1976年三木武夫
仏で第1回先進国首脳会議（サミット）。米上院公聴会でロッキード事件発覚。
1978年・1980年代福田赳夫
「日米防衛協力のための指針（ガイドライン）」を決定。
日米貿易摩擦問題化 → 対米自動車輸出規制(1981)。
1983年・1986年中曽根康弘
中曽根首相訪米、「日米運命共同体」「不沈空母」発言(1983)。
日米半導体協定で日本が輸出自主規制(1986)。
1988年・1989年竹下 / 海部
日米交渉で牛肉・オレンジの輸入自由化決定。
日米構造協議始まる(1989)。
1991年海部俊樹②
湾岸戦争 → 日、戦争支援90億ドル支出
1996年橋本龍太郎①
橋本龍太郎首相とクリントン大統領「日米安保共同宣言」。
沖縄・普天間基地の7年以内の返還決定。
1997年橋本龍太郎②
「新ガイドライン」決定 → 新ガイドライン関連法成立(1999)
2001年小泉純一郎①
米同時多発テロ事件 → テロ対策特別措置法を制定し自衛隊を後方支援に派遣
2003年
イラク戦争始まる → イラク復興支援特別措置法を制定し人道支援活動開始
重要事項と補足解説

  • アメリカはなぜ日本を開国させたかったのか？


    【産業革命の進展】
    飛躍的な発展を遂げたアメリカの紡績業は中国市場への進出を目指し、そのための太平洋横断航路の開設が企てられました。その航路上に日本が位置していたのです。
    【捕鯨業の発展】
    当時、アメリカは世界最大の捕鯨国であり、目的は灯油などに使う鯨油をとることにありました。漁場が太平洋へ移り日本近海での操業が急増したため、補給・避難のための港の提供を日本に期待しました。

  • ワシントン体制と東アジアの諸条約


    【桂・タフト協定 (1905年)】
    アメリカは日本の韓国指導権を承認し、日本はアメリカのフィリピン統治権を承認した覚書です。
    【石井・ランシング協定 (1917年)】
    アメリカは日本の中国に対する「特殊権益」を、日本はアメリカが主張する中国の「門戸開放・領土保全」を認めましたが、ワシントン会議の九カ国条約(1922年)締結に伴い廃棄されました。

  • 日米安全保障条約の変遷

    日米安全保障条約
    （1951年/吉田茂）    		 平和条約発効後の米軍駐留。極東有事と外国の対日攻撃の際の米軍出動（日本の再軍備を期待）。有効期限が明示されず。
    日米行政協定
    （1952年）    		 安保条約の細目協定。米軍への施設の無償提供、米軍人・家族の刑事裁判上の特権、防衛分担金の負担。
    新安全保障条約
    （1960年/岸信介）    		 米軍の日本防衛義務の明確化。軍事行動に関する事前協議制度（発議権は米国側）。日本領土内の共同防衛義務（「安保巻き込まれ論」）。条約有効期間は10年間。
    日米地位協定
    （1960年）    		 日米行政協定の継承と部分改正。米軍の基地外での権利の制限、防衛分担金の廃止。
    ガイドライン
    （1978年/福田赳夫）    		 正式名: 日米防衛協力のための指針。有事の際の日米共同作戦行動。
    新ガイドライン
    （1997年/橋本龍太郎）    		 日本の「周辺有事」での相互協力計画。1999年に周辺事態安全確保法などが成立。
